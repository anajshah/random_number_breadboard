\n For this project, I wanted to create a d8 that I could use when i played board games. I used a 7 segment display to show a random number from 0-7. Although it's a little too big for dnd I'm really happy with how it turned out.

## Parts Used
 - Keyestone Uno X 1
 - 7 semgent display X 1
 - Button X 1

## Final Project
The final prject displays a number from 0-7 on the screen when the button is pressed. I also found this really cool thing that scrambles the numbers for half a second before it shows you your number and I thought it looked really cool.

### Files
 - .ino - c++ code that controls the arduino
 - .brd - Electrical model of the breadboard circuit (made on tinkercad)
 - .csv - parts list
